### Install Dependencies

```
npm install
```

### Development

server start on `http://localhost:8080`

```
npm run dev
```

### Test

```
npm run test
```

### Build

```
npm run build
```

### Docker Compose

```
docker-compose -f docker-compose.production.yaml build
docker-compose -f docker-compose.production.yaml up
```

### Swagger

`http://localhost:8080/swagger-documentation`
